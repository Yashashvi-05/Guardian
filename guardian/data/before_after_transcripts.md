## How to Reproduce

```bash
python -m guardian.training.run_honest_episodes
# Episodes 1-9 are untrained (Phase 1), episodes 87-97 are trained (Phase 3)
# Read guardian/data/scorecards.jsonl to inspect raw reasoning traces
```

The improvement is not from tuned random numbers — Phase 3 uses a strictly better
algorithm (`_phase3_guardian` checks capability tags, cross-app signals, and full
action log history) vs. Phase 1 (`_phase1_guardian` checks only the last tool's type).
